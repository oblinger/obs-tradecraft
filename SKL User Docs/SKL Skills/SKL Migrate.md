# SKL Migrate (Skill: [[migrate/SKILL]])

Migrate an anchor — change its RID, traits, location, or structure.

- **Change RID** — rename all prefixed files, folders, wiki-links, config
- **Change location** — move folder, update HookAnchor, breadcrumbs, symlinks
- **Change traits** — add/remove traits, create/remove trait-required files
- **Move Claude session** — migrate `.claude/projects/` config to match new path
