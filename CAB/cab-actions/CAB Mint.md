
# /Mint

`/mint` -- Mint takes what's ready and makes it real. 
The main action. Read the feature doc or spec, execute the work, and checks result.  The agent works autonomously until the work is done or blocked.
- `/finalize` -- 

- **Mint** — 
- **Code** — write source code according to the spec.
- **Bugfix** — red test first, then fix. Mandatory for every bug encountered.
- **Spike** — aggressive root-cause diagnosis when stuck.
- **Forge** — rebuild, terminate, restart. Run the new code.
- **Test** — write and run tests. Verify the work matches the spec.
- **Review** — correctness check: anti-patterns, architecture.
- **Verify** — run full test suite, produce completion proof.

**Output:** Working result, passing tests, stat updated to "Done". Module docs and Files.md updated if source files changed.


(could be called render or launch)
